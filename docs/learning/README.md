### 2. [外部 API・サービス解説](./02-external-apis.md)

- **Google Sheets API**: スプレッドシートをデータベースとして活用
- **Resend API**: 開発者向けメール送信サービス
- **Vercel**: Next.js 最適化されたホスティング
- **料金体系**: 各サービスの無料枠と最適化

### 3. [プロジェクト構成とコード解説](./03-project-structure.md)

- **ファイル構成**: ディレクトリ構造と各ファイルの役割

- **API 設計**: RESTful API とエラーハンドリング
- **データフロー**: 状態管理とパフォーマンス最適化

### 4. [画像管理システムと管理機能](./04-image-management-system.md)

- **画像最適化**: Next.js Image コンポーネントの活用
- **モーダル表示**: アクセシビリティ対応のポップアップ機能
- **画像アップロード**: Base64エンコードとファイル検証
- **管理機能**: 動的なコンテンツ更新システム

### 5. [Vercel完全ガイド](./05-vercel-complete-guide.md)

- **自動デプロイ**: GitHubプッシュでの自動更新
- **環境変数管理**: セキュアな設定方法
- **メンテナンス**: 日常的な運用・監視方法
- **トラブルシューティング**: よくあるエラーと解決策

### 6. [セキュリティとエラーハンドリング](./06-security-error-handling.md)

- **セキュリティ対策**: 環境変数管理と入力値検証
- **エラーハンドリング**: 堅牢なエラー処理パターン
- **バリデーション**: 多層防御のデータ検証
- **デバッグ手法**: 効果的な問題解決方法

## 🎯 学習の進め方

### 初学者向け

1. **[フロントエンド技術](./01-frontend-technologies.md)** から始める
2. 各技術の公式ドキュメントを並行して読む
3. 小さなプロジェクトで実際に試してみる

### 中級者向け

1. **[プロジェクト構成](./03-project-structure.md)** で設計思想を理解
2. **[外部 API](./02-external-apis.md)** で実用的な統合方法を学ぶ
3. 自分なりのカスタマイズを加えてみる

## 💡 学習のポイント

### 🔧 実践的なスキル

- **モダン Web 開発**: 最新のベストプラクティス
- **型安全性**: TypeScript によるエラー削減
- **パフォーマンス**: 高速で快適なユーザー体験
- **保守性**: 長期的に維持しやすいコード

### 🚀 応用できる知識

- **フォーム処理**: バリデーション・送信・エラーハンドリング
- **外部 API 統合**: 認証・データ管理・メール送信
- **レスポンシブデザイン**: モバイル対応の UI/UX
- **デプロイメント**: 本番環境への公開

## 🎓 次のステップ

### 拡張アイデア

1. **ユーザー認証**: ログイン・会員機能の追加
2. **管理ダッシュボード**: 予約管理・統計表示
3. **決済システム**: Stripe 等での オンライン決済
4. **多言語対応**: 国際化（i18n）の実装
5. **PWA 化**: オフライン対応・プッシュ通知

### 類似プロジェクト

- **レストラン予約システム**
- **イベント管理システム**
- **ブログ・CMS**
- **EC サイト**
- **予約カレンダー**
