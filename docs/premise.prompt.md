# フルートレッスン予約LP プロジェクト要件書

## プロジェクト概要
フルート教師「吉原りえ」氏のレッスン予約システム付きランディングページの作成

## 参考サイト・リソース
- **デザイン参考**: https://lietoposto.com/
  - フルーティスト 吉原りえ氏の公式サイト
  - 清潔で上品なデザイン、スタジオ「Lieto Posto」の雰囲気を活用
- **既存Googleフォーム**: https://docs.google.com/forms/d/1P-zaPVq9gSXf5fL0eMqD7zJUtkFApZyChcscicJz9c8/viewform?edit_requested=true
- **参考実装**: https://script.google.com/macros/s/AKfycbytnUWbjsiUHiehqEj0NufieE_Nu_47Ox148qs9G-Vb9hDYMROBfVW-ir1uK-A6cc2qlg/exec

## 技術要件
- **フレームワーク**: Astro または Next.js
- **データベース**: Google スプレッドシート連携
- **メール送信**: 自動メール配信機能
- **レスポンシブ対応**: モバイル・タブレット・デスクトップ

## 機能要件
1. **ランディングページ構成**
   - ヘッダー（ナビゲーション）
   - メインビジュアル（講師紹介）
   - サービス紹介（レッスン内容・料金）
   - 予約フォーム
   - 講師プロフィール
   - アクセス情報
   - フッター

2. **予約システム機能**
   - Googleフォームの内容を完全再現
   - リアルタイムフォームバリデーション
   - 予約データのスプレッドシート保存

3. **自動メール配信**
   - 申し込み完了通知メール
   - 選択した支払い方法に応じた決済案内
   - 講師への通知メール

## デザイン要件
- **コンセプト**: 上品で親しみやすい、音楽の美しさを表現
- **カラーパレット**: lietoposto.comのデザインテイストを参考
- **フォント**: 読みやすく上品な日本語フォント
- **画像**: スタジオ・楽器・講師の写真を効果的に配置

## データ管理
- **予約データ**: Google スプレッドシートで一元管理
- **顧客情報**: プライバシー保護に配慮した安全な管理

## 今後の拡張性
- 予約カレンダー機能
- オンライン決済システム
- 生徒管理システム
