# Googleフォーム分析 - フルートレッスン予約システム

## 前提
元のGoogleフォーム: https://docs.google.com/forms/d/1P-zaPVq9gSXf5fL0eMqD7zJUtkFApZyChcscicJz9c8/viewform?edit_requested=true

## 想定されるフォーム項目（一般的なフルートレッスン予約フォーム）

### 1. 基本情報
- **お名前** (必須)
  - 姓・名の分離 または フルネーム
  - テキスト入力
- **ふりがな** (必須)
  - ひらがな入力
  - テキスト入力
- **メールアドレス** (必須)
  - 確認メール送信用
  - email形式バリデーション
- **電話番号** (必須)
  - 緊急連絡用
  - ハイフンあり/なし対応

### 2. レッスン希望情報
- **レッスン種別** (必須)
  - 選択肢（ラジオボタン）:
    - 初心者レッスン
    - 中級者レッスン
    - 上級者レッスン
    - 音大受験対策
    - 趣味・一般
- **レッスン形式** (必須)
  - 選択肢:
    - 個人レッスン
    - グループレッスン（2-3名）
    - オンラインレッスン
- **希望日時** (必須)
  - 第1希望: 日付 + 時間帯
  - 第2希望: 日付 + 時間帯
  - 第3希望: 日付 + 時間帯
- **レッスン時間** (必須)
  - 選択肢:
    - 30分
    - 45分
    - 60分
    - 90分

### 3. 音楽経験・レベル
- **フルート経験年数**
  - 選択肢:
    - 未経験
    - 1年未満
    - 1-3年
    - 3-5年
    - 5年以上
- **現在のレベル**
  - 選択肢:
    - 全くの初心者
    - 基礎を学習中
    - 簡単な曲が演奏できる
    - 中級レベル
    - 上級レベル
- **楽器の有無**
  - 選択肢:
    - 自分の楽器を持参
    - レンタル希望
    - 購入相談希望

### 4. 料金・支払い情報
- **料金プラン** (必須)
  - 選択肢:
    - 単発レッスン: ¥4,000/30分
    - 月2回コース: ¥7,000/月
    - 月4回コース: ¥13,000/月
    - 体験レッスン: ¥2,000/30分
- **支払い方法** (必須)
  - 選択肢:
    - 銀行振込
    - PayPay
    - 現金（当日支払い）
    - クレジットカード

### 5. その他
- **ご要望・ご質問**
  - テキストエリア（任意）
  - 特別な配慮が必要な事項
  - 学びたい曲やジャンル
- **どちらで知りましたか？**
  - 選択肢:
    - インターネット検索
    - SNS（Instagram/Twitter）
    - 友人・知人の紹介
    - チラシ・広告
    - その他

### 6. 確認・同意事項
- **キャンセルポリシーに同意** (必須)
  - チェックボックス
- **個人情報の取り扱いに同意** (必須)
  - チェックボックス

## 技術仕様

### バリデーション
- 必須項目チェック
- メールアドレス形式チェック
- 電話番号形式チェック
- 日付・時間の妥当性チェック

### データ保存
- Google Spreadsheet連携
- タイムスタンプ自動付与
- 重複申し込み防止

### メール送信
- 予約者への自動確認メール
- 講師への通知メール（オプション）
- 支払い方法別の案内テンプレート
